3 Observation COMPUTED

Every studio has its own plans. With 280 active membership plans across 57 studios, each studio averages ~4 unique plans. Naming is highly localized (e.g. “H - MEMBERSHIP : UNLIMITED AUTORENEW - Winnipeg”). The AI must reference the correct plan names per studio.
Program naming is inconsistent. 3,719 class types exist but only 288 are active. Multiple names refer to the same format (“Modo Yoga - 60 min”, “Modo 60”, “Historical Class60”). The knowledge base should normalize these.
